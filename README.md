# My Website Project

## Description
This repository contains a simple website project built using **HTML** and **Tailwind CSS**.  
It includes a **Home Page** and an **About Me Page**, demonstrating basic webpage structure, navigation links, and styling with Tailwind utility classes.

## Pages
- **Home Page (`index.html`)**: A simple landing page with a header, navigation, and featured content sections.
- **About Me Page (`about.html`)**: A personal introduction page with name, program, and interests.

## Features
- Responsive layout using **Tailwind CSS CDN**
- Clean navigation between pages
- Structured content using semantic HTML
- Easy to extend for additional pages or sections
